# AI & Machine Learning 

This project explores two major areas of machine learning: image classification and reinforcement learning.
- **Task 1** applies supervised learning to classify images using different algorithms.
- **Task 2** explores reinforcement learning through Grid World, implementing Value Iteration and Q-Learning.

Each section includes Jupyter notebooks and data files needed to reproduce the analysis.
